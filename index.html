Networking Concepts

  • DNS: The Domain Name System translates human-readable domain names (e.g., example.com) into IP addresses.
  • IP Addresses: Every device on the internet has a unique IP address used for communication.
  • Protocols: HTTP/HTTPS are protocols used to transfer data between a client and a server.
